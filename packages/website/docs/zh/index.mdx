## Sparkling 是什么？

Sparkling 是 TikTok 的跨平台混合容器，覆盖 **Android**、**iOS** 与 **[Lynx](https://lynxjs.org)**。
它帮助你将 Lynx bundles 渲染在原生壳中，并提供一致的强类型 JS ↔ 原生通信能力。
