## What is Sparkling?

Sparkling is TikTok's cross‑platform hybrid container spanning **Android**, **iOS**, and **[Lynx](https://lynxjs.org)**.
It lets you package Lynx bundles and render them inside native shells with a consistent
typed pipe for JS ↔ native communication.
